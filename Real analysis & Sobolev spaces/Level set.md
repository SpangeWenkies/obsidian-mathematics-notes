
# Definition: Level set

**Type:** #definition  

## Definition
- Let $f: X\mapsto \mathbb{R}$ be a function defined on a set $X$ (usually $\subseteq \mathbb{R}^n$)
- For $c\in \mathbb{R}$ the level set of $f$ at level $c$ is $$L_{c}(f)=\{ x \in X :f(x)=c \}$$
- Special cases
	- Sublevel set $\{ x:f(x)<c \}$
	- Superlevel set $\{ x:f(x)>c \}$ 

## Intuition (optional)
A level set is the set of points where the [[Graph]] of a function reaches a constant “height.”
