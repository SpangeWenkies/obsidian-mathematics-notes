
# Definition: Secant line

**Type:** #definition  

## Definition
- The secant line of a graph is for a function $f:\mathbb{R}\mapsto \mathbb{R}$ and points $x<y$ the line between $(x,f(x))$ and $(y, f(y))$ given by: $$L(t)=f(x)+\frac{f(y)-f(x)}{y-x}(t-x)$$
- This gives that a function is [[Convexity]] when: $$f(\lambda x+(1-\lambda)y)\leq \lambda f(x)+(1-\lambda)f(y), \lambda \in[0,1]$$
