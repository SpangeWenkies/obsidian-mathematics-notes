
# Definition: C space

**Type:** #definition  

## Definition
- $C^{k}$ and $C^{\infty}$
- Let $\Omega \subset \mathbb{R}^{n}$ be an [[Openness|open]] set 
- For integer $k\geq0$: $$C^{k}(\Omega)=\{ u:\Omega\mapsto \mathbb{R}|D^{\alpha}u \text{ exists and is continuous for all }|\alpha|\leq k \}$$

## Intuition (optional)
- $C^{0}(\Omega)$ is a [[Continuity|continuous]] function
- $C^{1}(\Omega)$ is a [[Continuous differentiability|continuously differentiable]] function
- $C^\infty(\Omega)$ is a [[Smooth function]], with [[Countability|countably]] infinite number of derivatives
