
# Definition: Multi-index

**Type:** #definition  

## Definition
- A multi-index is an [[Total order|ordered]] $n$-tuple of non-negative integers" $$\alpha=(\alpha_{1},\alpha_{2},\dots,\alpha_{n}), \ \ \ \alpha_{i}\in \mathbb{N}_{0}$$
- The order of $\alpha$ is $$|\alpha|=\sum_{i=1}^{n}\alpha_{i}$$
## Intuition (optional)
- Used as a way to track how many derivatives you take in each coordinate direction
- Used in definition of [[Weak derivative]] or partial derivatives in general.
