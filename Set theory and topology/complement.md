
# Definition: complement

**Type:** #definition  

## Definition
- insert here definition of a complement
- The complement of a set is a set, through the [[Principle of restricted comprehension]], which is through [[Axiom of replacement]]
	- Let $x\subseteq m$ then $m\setminus x:=\{ x \in m| x \not\in u \}$ for some $u$
