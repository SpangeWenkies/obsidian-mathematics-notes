
# Definition: functional relation

**Type:** #definition  

## Definition
- A [[relation]] $R$ is functional if $\forall x, \  \exists! \ y:R(x,y)$
- where $\exists!$ means to [[exist precisely]] 
- the [[Image]] of a functional relation $R$ is all $y$ for which $\exists x \in$ some set $m$ s.t. $R(x,y)$
