
# Definition: Axiom on union sets

**Type:** #definition  

## Definition
- Let $x$ be a set, then there exists a set whose elements are precisely the elements of the elements of $x$ 
- Notation: $u=\bigcup x$ is a set
- Generally:
	- Let $a_{1}.a_{2},\dots,a_{N}$ be sets
- Define recursively for all $N\geq3$ $$\{ a_{1},\dots,a_{N} \}:=\bigcup \{ \{ a_{1},\dots,a_{N-1} \},\{ a_{N} \} \}$$
	- For naive set theory this is only defined for $N$ finite
