
# Definition: real numbers

**Type:** #definition  

## Definition
- $\mathbb{R}:=^{\mathcal{A}}/_{\sim}$,
	where $\mathcal{A}$ is the set of [[almost homomorphism|almost homomorphisms]] on $\mathbb{Z}$ 

## Intuition (optional)
- We see we can construct it from the integers and the natural numbers, maybe also the rationals?

## Equivalent Formulations (optional)
- $\mathbb{R}:=\mathcal{P}(\mathbb{N})$
