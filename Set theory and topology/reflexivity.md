
# Definition: reflexivity

**Type:** #definition  

## Definition
- Let $M$ be a set, then a [[relation]] $\sim$ on elements of the set is reflexive if $$\forall m\in M: \ m \sim m$$ 
- note that this means $m\sim m$ is true
