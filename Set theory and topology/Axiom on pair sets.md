
# Definition: Axiom on pair sets

**Type:** #definition  

## Definition
- Let $x,y$ be sets, then there exists a set that contains, as its elements, precisely the sets $x$ and $y$, i.e., $$\forall x:\forall y: \exists m : \forall u: (u \in m \iff u=x \ \lor \ u=y)$$
- The notation: we denote the set $m=\{ x,y \}$
- This also gives the existence of the set with one element, e.g., $\{ x \}$
	- $\{ x \}:=\{ x,x \}$
	- There is no more info in it than $x$

## Intuition (optional)
- Is $\{ x,y \}=\{ y,x \}$? $\implies$ yes!
	- $a \in \{ x,y \}\implies a\in \{ y,x \}$
	- $a\in \{ y,x \}\implies a\in \{ x,y \}$
	- $\implies \{ x,y \}\subseteq \{ y,x \} \ \wedge \ \{ y,x \}\subseteq \{ x,y \}$
	- $\implies\{x,y\}=\{ y,x \}$ by definition of [[equality]]
