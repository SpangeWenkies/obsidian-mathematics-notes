
# Definition: Pre-image

**Type:** #definition  

## Definition
- Let $\phi:A\to B$ be any [[Map]]
- then the set $preim_{\phi}(V)$ for $V\subseteq B$ is defined as $$preim_{\phi}(V)=\{ a\in A|\phi(a)\in V \}$$
- The pre-image is itself a set by [[Principle of restricted comprehension]]
