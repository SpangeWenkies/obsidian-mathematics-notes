
# Definition: Inverse

**Type:** #definition  

## Definition
- For a [[Map]] $\phi:A\to B$, being a [[Bijection]], the inverse of map $\phi$ is $\phi^{-1}:B\to A$
- This is defined uniquely by $\phi^{-1}\circ\phi=Id_{A}$ and $\phi \circ \phi^{-1}=Id_{B}$, where $Id$ denote the [[Set identity]]
