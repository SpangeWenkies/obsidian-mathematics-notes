
# Definition: (n in N)-th power map

**Type:** #definition  

## Definition
- $S^{n}:=S\circ S^{P(n)}$ 
	if $n\in N\setminus \{ 0 \}$ 
	and $S^{0}:=id_{\mathbb{N}}$, 
	where $id_{\mathbb{N}}$ is the [[Set identity]] of $\mathbb{N}$
