
# Definition: Transitivity

**Type:** #definition  

## Definition
- Let $M$ be a set, then a [[relation]] $\sim$ on elements of the set is transitive if $$\forall m,n,p \in M:m\sim n \land n\sim p \implies m\sim p$$
