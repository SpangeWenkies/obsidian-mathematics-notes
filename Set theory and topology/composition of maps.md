
# Definition: composition of maps

**Type:** #definition  

## Definition
- If given two maps $A\to^{\phi}B, B\to^{\psi}C$, we construct $\psi \circ \phi$
- Writing this into a diagram of all [[Map|maps]] , makes us able to see if the diagram [[Commutability|commutes]] 
- Composition is [[associative]] , i.e., if we have some other map $\xi$ , then $\xi \circ(\psi \circ\phi)=(\xi \circ \psi)\circ\phi$
