
# Definition: Image

**Type:** #definition  

## Definition
- For a [[Map]] $\phi:A\to B$, $a\mapsto \phi(a)$, the image $im_{\phi}(A)\equiv\phi(A):=\{ \phi(a) \in B|a\in A \}$
- We can actually check that this definition adheres to the [[Principle of restricted comprehension]]
