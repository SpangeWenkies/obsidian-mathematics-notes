
# Definition: predecessor map

**Type:** #definition  

## Definition
- $P:\mathbb{N}\setminus \{ 0 \}\to \mathbb{N}, n\mapsto m:m\in n$
- e.g. $P(2)=P(\{ \{ \emptyset \} \})=\{ \emptyset \}=1$
