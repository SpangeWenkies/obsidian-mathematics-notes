
# Definition: Natural numbers

**Type:** #definition  

## Definition
- By [[Axiom of infinity]] it constitutes a set
- The axiom for the simplest case of $\emptyset$ gives the following set $$\{ \emptyset,\{ \emptyset \},\{ \{ \emptyset \} \},\{ \{ \{ \emptyset \} \} \},\dots \}:={0,1,2,3,\dots}=\mathbb{N}$$
