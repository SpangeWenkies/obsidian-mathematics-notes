# Definition: linear map / operator

**Type:** #definition  

## Definition
- if a map $A:\mathcal{D}_{A}\to\mathcal{H}$, where $\mathcal{H}$ is some set, has that $\mathcal{D}_{A}\subseteq\mathcal{H}$ 
	then $A$ is a linear map
		here $\mathcal{D}_{A}$ is the [[Domain]] of map $A$
