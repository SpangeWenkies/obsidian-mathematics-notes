
# Definition: coinciding linear maps

**Type:** #definition  

## Definition
- for a [[linear map]] $A$ and a linear map $B$ we have that they coincide if the following hold:
	- $\mathcal{D}_{A}=\mathcal{D}_{B}$ (equal [[Domain]])
	- $A\psi=B\psi,$ where $\psi \in \mathcal{D}_{A}=\mathcal{D}_{B}$

## Intuition (optional)
- "they do the same operation"
