
# Definition: Lebesgue decomposition

**Type:** #definition  

## Definition
- We can decompose a [[Measure]] $\nu$ into a unique decomposition w.r.t. some other measure $\mu$ by $$\nu=\nu_{ac}+\nu_{s},$$ where $\nu_{ac}\ll \mu$ and $\nu_{s}\perp \mu$
