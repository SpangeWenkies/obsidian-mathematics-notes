
# Definition: Martingale property

**Type:** #definition  

## Definition
$$E_{\mathbb{P}}[[Y_{t}-Y_{s}]_{\chi_{F}}]=0,\text{ where }\chi_{F}\text{ is an indicator function of event }F$$ $$\iff$$ $$Y_{s}=E_{\mathbb{P}}[Y_{t}|\Sigma_{S}]$$
