
# Definition: π-system

**Type:** #definition  

## Definition
- A $\pi$-system is a collection of sets [[Closure|closed under]] finite intersections

## Intuition (optional)
- e.g. $A,B \in \pi \implies A \cap B\in \pi$
- e.g. intervals in $\mathbb{R}$, rectangles $A\times B$, [[Cylinder set]], [[Ball]], [[Sigma algebra|algebras of sets]]
