
# Definition: Sub-markov kernel

**Type:** #definition  

## Definition
- Where we have that $$\text{if }\forall x \in X, K(x,X)=1\text{ defines a Markov kernel (prob.) }$$
	we have that $$\text{if }\forall x \in X, K(x,X)\leq 1\text{ defines a sub-markov kernel}$$
	

## Intuition (optional)
- The sub-markov kernel can assign less than 100% probabillity to stay inside the [[State space]]
- The [[Markov process]] can have a [[Killing]] or a [[Jump]] 

- [[Resolvant|Resolvants]], denoted by $G_{\alpha}$, are sub-markovian [[Operator|operators]]
