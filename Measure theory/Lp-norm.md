
# Definition: Lp-norm

**Type:** #definition  

## Definition
- the Lp-norm of a function $f$ is denoted by:
	- for $p<\infty$: $||f||_{p}=\left( \int|f|^{p}d\mu \right)^{1/p}$
	- for $p\to \infty$: $||f||_{\infty} = \inf\{ M: |f(x)|\leq m\text{ a.e.} \}$
